# 📖 Литари

> *«Рукописи не горят»* — Михаил Булгаков

**Литари** — интерактивная платформа для глубокого погружения в книги. Карты локаций, граф персонажей, иллюстрации и живое читательское сообщество — всё в одном месте.

---

## Что это такое

Чтение сложной книги — особенно с десятками персонажей и переплетёнными сюжетными линиями — часто превращается в борьбу с собственной памятью. Литари решает эту проблему: платформа превращает пассивное чтение в интерактивное исследование.

Первый проект — **«Мастер и Маргарита»** Булгакова: роман с тремя мирами, двумя временными пластами и галереей незабываемых героев.

```
Чтение + визуализация + общение = новый уровень погружения в книгу
```

## Функциональность

###  Карта локаций
Все локации романа — от Патриарших прудов до Бала у Сатаны — собраны на одной визуальной карте. Три мира романа Москва 1930-х, Ершалаим, потусторонний мир разделены визуально.

### Граф персонажей
Интерактивная карта связей между героями: любовь, подчинение, конфликт, предательство, союз. Нажмите на персонажа — и увидите его роль, описание и цитату. Позволяет видеть всю сеть отношений одним взглядом.

### Иллюстрации
Медиатека по главам: кадры из экранизаций, авторские иллюстрации, художественные интерпретации. Удобная фильтрация по главам — от «При свечах» до «Великого бала у Сатаны».

###  Форум
Структурированное пространство для обсуждений: теории, разбор сцен, сравнение экранизаций, открытые вопросы. Никаких разрозненных чатов — одно место для всех, кто думает о книге.
